# Complete MDX Features Guide

Welcome to our comprehensive guide to MDX features! This post demonstrates all the available formatting and styling options you can use in your blog posts.

## Text Formatting

You can make text **bold**, _italic_, or **_both_**. You can also use ~~strikethrough~~ for deleted text.

### Inline Code

### Code Block with Filename

```python filename="hello.py"
def greet(name: str) -> str:
    """A simple greeting function"""
    return f"Hello, {name}!"

# Call the function
result = greet("Linux User")

```

### Highlighted Lines

```javascript {3,5-7} filename="example.js"
function calculateTotal(items) {
  return (
    items
      .filter((item) => item.inStock) // Only in-stock items
      .map((item) => item.price)
      // Calculate total with tax
      .reduce((total, price) => {
        return total + price * 1.2; // 20% tax
      }, 0)
  );
}
```

## GitHub-style Alerts

You can use GitHub-style alerts to highlight important information:

<Alert type="note">
  This is a note alert. Use it to highlight information that users should take
  into account, even when skimming.
</Alert>

<Alert type="tip">
  This is a tip alert. Perfect for optional information to help users be more
  successful.
</Alert>

<Alert type="important">
  This is an important alert. Use it for crucial information necessary for users
  to succeed.
</Alert>

<Alert type="warning">
  This is a warning alert. Critical content demanding immediate user attention
  due to potential risks.
</Alert>

<Alert type="caution">
  This is a caution alert. Use it to advise about negative potential
  consequences of an action.
</Alert>

## Advanced Features

### Custom Components

You can use React components directly in your MDX:

```jsx filename="example.tsx"
<CodeBlock
  language="typescript"
  filename="example.tsx"
  code={`
    const greeting = "Hello, World!";
    console.log(greeting);
  `}
  highlightLines={[2]}
/>
```

### Keyboard Shortcuts

Use `Ctrl + C` to copy and `Ctrl + V` to paste.
